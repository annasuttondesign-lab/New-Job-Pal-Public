Welcome to New Job Pal

Let's get your AI-powered job search companion up and running.

This guide walks you through every step. No experience needed — each command is explained, and you can check off steps as you go.

How to read these instructions
example command Copy

Dark boxes like this are terminal commands. Click Copy, paste into your terminal, and press Enter. Each command tells your computer to do something — download a file, install a tool, start the app. This is how you'll lead your computer through the setup, one step at a time.

Explanation boxes tell you what a command does in plain English.

You don't need to do anything with these — they're just context so you know what's happening before you run something.

What's a terminal? It's the app where you type commands to talk to your computer. On Mac, it's called Terminal (search for it in Spotlight with Cmd + Space). On Windows, use Command Prompt or PowerShell (search from the Start menu). It looks like a window with a blinking cursor — that's where you'll paste the commands from this guide.
A note on safety: It's smart to be careful about pasting commands you find online. Every command in this guide is standard, widely-used, and open source — npm install downloads libraries from the public npm registry, and npm start runs the app locally on your machine. The app's own code is right here in the project for you to read, and its dependencies (listed in package.json) are widely-used open source libraries downloaded from the public npm registry.

1
Install Node.js
The engine that runs the app

Node.js lets your computer run JavaScript outside of a browser. New Job Pal's server is built with it.

Go to nodejs.org and download the LTS (Long Term Support) version. Run the installer and accept the defaults.

Once installed, verify it works by opening your Terminal (Mac) or Command Prompt (Windows) and running:

node --version
What this does: Prints the version of Node.js you installed (e.g., v20.11.0). If you see a version number, you're all set.
Having trouble?

"command not found" — Close your terminal and reopen it. The installer needs a fresh terminal to register.

Mac users: If you use Homebrew, you can also install with brew install node.

Need more help? The official Node.js download page has guides for every operating system.

2
Download the Project
Get the code onto your computer

If you downloaded this as a ZIP file, unzip it and open a terminal in that folder. If you're using Git, clone the repository:

git clone https://github.com/yourusername/new-job-pal.git

Then navigate into the project folder:

cd new-job-pal
What cd does: "Change directory" — it moves your terminal into the project folder so the next commands know where to look.

Now install the dependencies (libraries the app needs):

npm install
What this does: Reads the package.json file and downloads everything the app needs into a node_modules folder. This may take a minute.
Having trouble?

"npm: command not found" — npm comes bundled with Node.js. Go back to Step 1 and make sure Node installed correctly.

Permission errors on Mac/Linux: Try sudo npm install and enter your password. (Better long-term fix: fix npm permissions.)

3
Get Your API Key
The key that powers the AI features

New Job Pal uses Claude (by Anthropic) for its AI features — resume generation, cover letters, mock interviews, and more. You'll need an API key to connect.

  1. Go to console.anthropic.com
  2. Create a free account (or sign in)
  3. Add a payment method under Billing (AI usage is pay-as-you-go; typical use costs pennies)
  4. Go to Settings → API Keys and click Create Key
  5. Copy the key — it starts with sk-ant-
Keep it secret! Your API key is like a password. Never share it publicly or commit it to Git. The .env file you'll create next is already in .gitignore, so it won't be uploaded.
Having trouble?

Don't want to set up billing yet? The app will still run! You just won't be able to use AI features (resume generation, mock interviews, etc.) until you add a key. Everything else works fine.

4
Create Your .env File
Where your private settings live
What's a .env file? It's a simple text file that stores private settings (like API keys) that your app reads on startup. The dot at the beginning makes it a hidden file on Mac/Linux. It's already in .gitignore, so it will never be uploaded to GitHub.

The project includes a template called .env.example. The fastest way to set up is to copy it in your terminal:

cp .env.example .env
What this does: Creates a new file called .env by copying the template. The template already has the right format — you just need to paste in your real API key.

Now open .env in any text editor (VS Code, Notepad, TextEdit — anything works). You'll see this:

.env 
ANTHROPIC_API_KEY=your-api-key-here
PORT=3000

Replace your-api-key-here with the real API key you got in Step 3 (the one starting with sk-ant-). Leave PORT=3000 as-is. Save the file.

Prefer to create the file manually instead?

Option A — From your text editor: Open VS Code (or any editor), create a new file, paste in the two lines above (with your real key), and save it as .env in the project's root folder (the same folder where package.json lives). Make sure the file is named exactly .env with no .txt extension.

Option B — From the terminal on Mac/Linux:

echo "ANTHROPIC_API_KEY=sk-ant-YOUR-KEY-HERE" > .env && echo "PORT=3000" >> .env

Replace sk-ant-YOUR-KEY-HERE with your actual key.

Having trouble?

Can't see the file after creating it? Files starting with a dot are hidden by default. On Mac Finder, press Cmd + Shift + . to toggle hidden files. On Windows, enable "Show hidden files" in File Explorer options.

Windows users: The cp command might not work in Command Prompt. Use copy .env.example .env instead, or just create the file manually with Notepad.

5
Launch New Job Pal
The moment of truth

Start the server:

npm start
What this does: Starts the New Job Pal server. You should see a message like Job Pal running on http://localhost:3000 — that means it's working!

Open your browser and go to:

http://localhost:3000

You should see the New Job Pal dashboard. Start by heading to the Profile section to enter your skills and experience — the AI uses this to personalize everything it generates for you.

Having trouble?

"Error: Cannot find module" — You may have skipped npm install. Go back to Step 2 and run it.

"Port 3000 already in use" — Another app is using that port. Change PORT=3000 to PORT=3001 in your .env file and try again.

To stop the server: Press Ctrl + C in the terminal where it's running.
